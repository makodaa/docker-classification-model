Image Classification as a Service

Description

The following project is a full stack application for an image classification service for invasive plants.

  • Frontend - Written in Vanilla JS, provides an interface for image upload and classification
  • Backend - Written in Python, loads the image classification model and the database, and provides Flask API endpoints for the frontend.
  • Database - A PostgreSQL database which stores classification result history.

Each layer of the full stack application is containerized using Docker. Each container holds the dependencies of that layer, isolated from what the machine currently has.
